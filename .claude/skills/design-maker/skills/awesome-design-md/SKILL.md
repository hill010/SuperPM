# Awesome DESIGN.md

A curated collection of DESIGN.md files extracted from real websites. Copy a DESIGN.md into your project and tell your AI agent "build me a page that looks like this" to get pixel-perfect UI.

## What is DESIGN.md?

[DESIGN.md](https://stitch.withgoogle.com/docs/design-md/overview/) is a plain-text design system document introduced by Google Stitch. It's a markdown file that AI agents read to generate consistent UI.

| File | Who reads it | What it defines |
|------|-------------|-----------------|
| `AGENTS.md` | Coding agents | How to build the project |
| `DESIGN.md` | Design agents | How the project should look and feel |

## Usage

1. Visit https://getdesign.md/ to browse available design files
2. Copy the DESIGN.md content for your chosen platform
3. Paste it into your project as `DESIGN.md`
4. Tell your AI: "Build me a page following this DESIGN.md"

## Tips for Best Results

1. **Be specific** - Reference specific sections from the DESIGN.md
2. **Adapt don't copy** - Use the design system as inspiration, adjust for your content
3. **Maintain consistency** - Apply the same design tokens throughout your project
4. **Check accessibility** - Ensure color contrasts meet WCAG guidelines
5. **Test responsive** - Most DESIGN.md files include mobile breakpoints
