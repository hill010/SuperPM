# Product Spec

## 产品概述

AI SplitMaster 是一个面向 AI 视频与视觉创作的对话式创作工作台。产品通过多 Agent 协作，把用户的故事创意、素材生成需求或角色设定，逐步转化为可审核、可复用、可导出的创作产物。

当前原型的核心不是传统表单式流程，而是 **左侧对话驱动 + 右侧画布沉淀产物**：

- 左侧负责：输入需求、Agent 对话、参数选择、阶段确认、修改反馈。
- 右侧负责：展示已经生成或确认过的阶段产物。
- 关键原则：右侧画布不能提前展示未来产物，所有内容必须由左侧当前阶段生成或确认后，按顺序出现。

**核心价值**：

- 降低 AI 视频创作门槛，让用户用对话方式推进创作。
- 把剧本、角色、场景、分镜、视频、音效等复杂流程拆成可确认节点。
- 把生成结果沉淀成项目资产，支持故事成片、自由素材生成和角色资产复用。

**目标用户**：

| 用户类型 | 核心诉求 | 产品价值 |
|---|---|---|
| 小白创作者 | 不懂分镜和提示词，也想把创意变成视频 | 对话式引导、阶段确认、低门槛生成 |
| AI 创作者 | 已使用 AI 生图/视频工具，需要统一工作流 | 标准化流程、资产复用、多模型参数管理 |
| 短视频/动画创作者 | 需要快速沉淀剧本、角色、场景、分镜和成片 | 从故事到视频的完整创作链路 |
| 角色/世界观创作者 | 需要稳定角色设定和可复用角色资料 | 角色形象、概念图、三视图和设定细节沉淀 |

## 产品形态

当前原型以单文件 HTML 验证交互，后续产品形态仍以桌面端为主。

| 版本 | 模式 | 说明 |
|---|---|---|
| V1 原型/MVP | 本地工作台 + 用户自带 API | 验证三大核心工作流、对话驱动画布、项目与资产管理 |
| V2 | 平台算力 + 云端资产 | 增加账号、算力余额、充值、云端项目协作和素材中心 |

## 核心交互原则

### 0. OiiOii 产品参考原则

当前产品原型需要参考 `AI_SplitMater/OiiOii使用手册.docx` 中的产品设计与交互细节，但不做一比一复制。可借鉴的核心模式如下：

- **按目标选择入口**：用户不是先理解复杂功能，而是根据“我要做什么”选择入口，例如快速生成素材、制作剧情短片、设计角色资产。
- **工作台即主创作空间**：进入具体模式后，用户在工作台内通过左侧对话完成后续操作，不频繁跳转页面。
- **自然语言优先**：提示词不要求专业格式，用户可以用自然语言描述故事、画面、角色或修改要求。
- **Agent 接力推进**：不同阶段由不同 Agent 加入对话，说明当前任务、生成结果、等待用户确认。
- **生成后再编辑**：用户先生成结果，再通过对话、悬浮操作、编辑按钮、历史记录等方式修改和筛选。
- **资产沉淀与复用**：满意的角色、场景、图片、视频、音效可以加入资产库，在后续故事成片或自由画布中复用。
- **创作过程可回退/重选**：对图片、视频、角色主图等生成结果，需要保留重新生成、修改描述、下载、历史记录、确认选择等入口。
- **消耗可感知**：后续如果接入平台算力，需要在生成前后展示“盒饭/算力”消耗参考和余额。

### 1. 对话驱动

用户通过左侧对话框输入创作意图、补充要求、提出修改。Agent 根据当前阶段给出任务说明、生成建议和确认入口。

### 2. 参数在左侧选择

所有模型、风格、资产、比例、分辨率、角色风格等设置都在左侧对话框下方完成。右侧画布不承担参数配置。

### 3. 右侧画布只沉淀结果

右侧画布只显示已经生成或确认过的内容。任何未来阶段的结果，在未触发当前阶段生成/确认前不能出现。

### 4. 阶段必须按序推进

用户需要先完成当前阶段的输入、选择和生成，再确认是否满意。确认后才进入下一阶段。

### 5. 产物可复用

角色、场景、分镜、视频、音效、自由画布素材都应作为项目资产沉淀，后续可在其他工作流中引用。

## 核心工作流总览

当前原型包含三个主工作流：

| 工作流 | 入口 | 目标产物 | 核心特点 |
|---|---|---|---|
| 故事/剧情成片 | 首页「故事/剧情成片」 | 最终成品视频 | 从故事创意到剧本、角色、场景、分镜、视频、音效、成片 |
| 自由画布 | 首页「自由画布」 | 图片/短视频素材 | 快速输入提示词，选择模型和参数后生成素材 |
| 角色设计 | 首页「角色设计」 | 可复用角色资产 | 从角色信息到形象图、概念图、三视图和设定细节 |

## 工作流一：故事/剧情成片

**目标**：用户输入故事创意后，系统通过多 Agent 协作，最终得到一个视频成品预览。

### 阶段流程

| 顺序 | 阶段 | Agent | 用户动作 | 右侧画布产物 |
|---|---|---|---|---|
| 1 | 基础设置 | 艺术总监 | 确认片长、画幅、对白语言、情绪基调 | 项目基础设置 |
| 2 | 剧本生成 | 编剧 | 确认或修改剧本方向 | 剧本大纲、场景节拍、对白基础 |
| 3 | 角色设计 | 角色设计师 | 确认角色卡和角色图 | 角色卡、主图、三视图 |
| 4 | 场景设计 | 场景设计师 | 确认场景设定 | 场景主图、氛围描述、多视图参考 |
| 5 | 分镜方案 | 分镜师 | 确认分镜策略和视频模型 | 分镜方案、生成策略 |
| 6 | 宫格分镜 | 分镜师 | 审核镜头顺序 | 静帧分镜宫格 |
| 7 | 分镜视频 | 分镜师 | 确认逐镜头视频片段 | 分镜视频片段 |
| 8 | 音效设计 | 剪辑师（音效） | 确认剪辑、配乐、环境音和音效 | 音效方案、剪辑建议 |
| 9 | 最终成片 | 剪辑师 | 预览或导出 | 最终成品视频 |

### 关键规则

- 每个阶段由对应 Agent 负责说明任务、生成产物、等待用户确认。
- 用户确认当前阶段后，才进入下一阶段。
- 最终成片必须在右侧画布展示视频预览。
- 剪辑师（音效）阶段包含分镜视频生成、剪辑节奏、配乐、环境音和关键音效。
- 剧本、角色、场景、分镜等产物应支持两种修改方式：
  - 左侧对话框直接输入修改要求。
  - 右侧画布展开产物后进行局部编辑或重新生成。
- 分镜阶段需要支持两类生成策略：
  - 多图参考生成：基于角色图、场景图和分镜描述直接生成。
  - 宫格图生成：根据镜头复杂程度生成四宫格或九宫格视图，用户确认后再生成视频。
- 分镜视频需要支持悬浮操作：重新生成、剪辑、修改、下载、查看历史记录。
- 最终成片前应支持“合成预览”，确认后再进入最终视频更新或导出。

## 工作流二：自由画布

**目标**：用户快速生成图片或短视频素材，不进入完整故事成片流程。

### 操作流程

1. 用户从首页切换到「自由画布」。
2. 系统自动进入工作台并创建自由画布工作空间。
3. 左侧对话区提醒用户可以快速生成图片或视频素材。
4. 用户在左侧输入框输入素材提示词。
5. 用户在左侧参数区选择：
   - 生图模型
   - 生成类型
   - 风格参考
   - 资产引用
   - 画幅比例
   - 分辨率
6. 用户点击「生成」。
7. 右侧画布才出现生成结果。

### Agent 阶段

| 阶段 | Agent | 说明 |
|---|---|---|
| 自由画布 | 画布助手 | 理解创作意图、参考素材和输出目标 |
| 提示词参数 | 提示词助手 | 整理主体、风格、镜头、参数和负面词 |
| 图像生成 | 图像生成器 | 生成单图、变体或局部重绘 |
| 短视频素材 | 短视频生成器 | 生成短视频素材并沉淀到画布 |

### 关键规则

- 所有设置项都在左侧对话框下方选择。
- 右侧画布只显示等待状态或已生成素材。
- 未点击「生成」前，右侧不能提前展示生成结果。
- 自由画布可从首页进入，也可以从故事成片等生成链路中通过「添加画板」进入。
- 用户可输入自然语言提示词，也可上传参考图作为风格或画面参考。
- 生成后，素材节点应支持：
  - 快速生成/继续生成
  - 修改描述
  - 下载保存
  - 右键多选或框选多个素材
  - 使用多个素材继续生成图片或视频
- 用户可以在画布对话框中引用已有素材或资产，继续生成新图片/视频。

## 工作流三：角色设计

**目标**：用户输入角色信息后，生成可复用的完整角色资产。

### 最新流程

| 顺序 | 阶段 | Agent | 用户动作 | 右侧画布产物 |
|---|---|---|---|---|
| 1 | 用户输入角色信息 | 用户 | 输入角色身份、性格、外观、服装、世界观等 | 无，等待工作流激活 |
| 2 | 工作流规划 | 艺术总监 | 点击「激活角色工作流」 | 工作流规划节点 |
| 3 | 角色形象生成 | 角色设计师 | 选择视觉风格，点击「生成角色形象图」 | 角色信息分析 + 角色形象图 |
| 4 | 角色概念图设计 | 角色设计师 | 确认形象满意后，点击「生成角色概念图」 | 角色概念图 |
| 5 | 角色全量资料 | 角色设计师 | 确认概念图满意后，点击「生成全量资料」 | 三视图、角色设定细节、完整资料图 |

### 阶段细节

#### 1. 用户输入角色信息

用户在首页或工作台左侧输入角色描述，例如：

- 身份
- 性格
- 外观
- 服装
- 世界观
- 能力设定
- 情绪气质

此时右侧画布不展示角色结果。

#### 2. 艺术总监激活工作流

艺术总监介入，理解角色信息并规划生成任务。

用户点击「激活角色工作流」后，右侧画布才出现工作流规划节点。

#### 3. 角色形象图

角色设计师介入，对角色信息进行分析。用户需要选择角色视觉风格，例如：

- 动画写实
- 国漫
- 赛璐璐

用户点击「生成角色形象图」后，右侧画布生成：

- 角色信息分析
- 初始形象要点
- 角色形象图

生成完成后，用户需要确认是否满意。

#### 4. 角色概念图

用户确认角色形象满意后，进入概念图设计。

用户点击「生成角色概念图」后，右侧画布生成：

- 概念方向 A
- 概念方向 B
- 角色概念图

生成完成后，用户需要再次确认是否满意。

#### 5. 角色全量资料

用户确认概念图满意后，角色设计师基于前面已确认的内容延展完整资料。

右侧画布生成：

- 三视图
- 表情/动作参考
- 角色设定细节
- 一致性锁定信息
- 完整资料图

### 关键规则

- 角色设计最容易出错的规则是：右侧画布必须按步骤出现内容。
- 未激活工作流前，不出现规划节点。
- 未选择视觉风格并生成前，不出现角色形象图。
- 未确认形象图满意前，不出现概念图。
- 未确认概念图满意前，不出现全量资料。
- 如果当前阶段还没生成产物，用户点击「满意，继续」时应提示先生成并确认当前阶段产物。
- 角色主图生成后，需要支持用户筛选和确认最满意版本。
- 角色主图应支持悬浮操作：重新生成、添加到左侧对话继续修改、修改描述、下载。
- 角色主图和概念图应支持历史记录，用户可以从历史版本中选择满意结果并确认。
- 满意的角色应支持加入角色资产库，并在自由画布或故事成片工作流中调用，保证同一角色脸部和特征一致。

## 项目管理能力

### 首页与全局导航参考

参考 OiiOii 使用手册，首页与全局导航需要承载以下能力：

- 首页顶部：语言切换、登录/注册、账户入口、会员/算力入口。
- 登录后：显示资产、会员/盒饭、活动、个人中心等入口。
- 左侧导航：切换首页、项目、资产库、回收站等模块。
- 发现/灵感社区：可作为后续版本功能，用于浏览优质作品和创作流程。
- 功能亮点入口：用目标场景引导用户选择正确工作流，而不是让用户理解底层功能。

### 分享项目

用户可从工作台顶部打开分享项目弹窗。

功能包括：

- 生成项目分享链接
- 设置分享权限
  - 只读查看
  - 允许评论
  - 协作编辑
- 复制分享链接

### 项目设置

用户可从更多菜单打开项目设置弹窗。

功能包括：

- 项目重命名
- 盒版/工作流使用设置
- 项目资源概览
- 删除项目危险操作入口

## 资产体系

资产是跨工作流复用的核心能力。

### 资产类型

| 类型 | 来源 | 用途 |
|---|---|---|
| 角色资产 | 角色设计、故事成片角色阶段 | 自由画布引用、故事分镜复用、视频生成一致性 |
| 场景资产 | 故事成片场景阶段 | 分镜生成、自由画布生图、视频背景 |
| 图片素材 | 自由画布、分镜生成 | 素材库、图生图、视频生成参考 |
| 视频素材 | 分镜视频、自由画布短视频 | 剪辑、最终成片、素材复用 |
| 音效素材 | 音效设计阶段 | 最终成片、声音资产库 |

### 资产复用交互

参考 OiiOii 的资产使用方式，资产复用需要覆盖：

- 在首页或工作台中进入「我的资产」。
- 在左侧悬浮导航或资产入口中快速打开资产库。
- 角色资产支持“用 Ta 出演”或类似快捷入口。
- 在自由画布或故事成片流程中，可直接引用角色/场景资产。
- 被引用资产应帮助保持角色脸部、服装、场景风格和画风一致性。
- 生成结果满意后，可下载到本地或加入资产库。

## 页面定义

### 首页

首页负责承接用户创作意图，并选择工作流。

核心元素：

- 创作输入框
- 工作流切换：
  - 故事/剧情成片
  - 自由画布
  - 角色设计
- 快捷入口卡片
- 最近项目/项目列表

关键交互：

- 选择「故事/剧情成片」后，输入故事创意进入完整成片流程。
- 选择「自由画布」后，自动进入工作台并创建自由画布工作空间。
- 选择「角色设计」后，输入角色信息进入角色设计工作流。

### 工作台

工作台是产品核心页面。

| 区域 | 内容 |
|---|---|
| 左侧导航 | 返回首页、新建项目、工作流入口 |
| 左侧对话区 | 当前 Agent、对话流、任务卡、输入框、参数选择区 |
| 右侧画布区 | 阶段步骤条、已生成产物、等待状态、画布工具 |
| 顶部项目操作 | 分享项目、更多菜单、项目设置 |

核心规则：

- 左侧驱动生成，右侧展示结果。
- 参数选择在左侧，不放在右侧画布。
- 右侧画布只保留阶段定位和产物展示。
- 右侧画布内容必须按用户确认顺序出现。

### 设置页

用于配置 AI 服务和 API Key。

V1 重点：

- 文本 AI 服务商配置
- 图像 AI 服务商配置
- API Key 验证状态
- 本地加密保存

### 模板库

用于复用预设工作流模板。

当前优先级低于三大核心工作流。

### 资产库

用于管理用户生成和上传的资产。

当前重点是支持角色、场景、图片、视频素材的复用入口。

### 导出中心

用于导出项目产物。

优先支持：

- Markdown
- JSON
- 视频预览/交付包

## 功能优先级

## 当前原型验收重点

继续推进原型时，优先检查以下问题：

1. 三个首页入口是否都能进入正确工作流。
2. 左侧参数区是否随工作流和阶段变化。
3. 右侧画布是否只展示当前已生成/已确认产物。
4. 角色设计流程是否严格遵循：
   - 输入角色信息
   - 艺术总监激活工作流
   - 生成角色形象图
   - 确认后生成角色概念图
   - 确认后生成全量资料
5. 自由画布是否必须点击「生成」后才显示素材结果。
6. 故事/剧情成片是否最终展示视频成品。
7. 生成后的图片/视频/角色主图是否具备重新生成、修改、下载、历史记录等入口。
8. 满意的角色、场景和素材是否能加入资产库并被其他工作流复用。
