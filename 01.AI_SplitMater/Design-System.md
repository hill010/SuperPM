## 7. 设计原则

### 7.1 编辑型设计风格

- **浅灰大面积背景**：#F4F4F2 作为页面主体，营造阅读感
- **Hero深色渐变**：蓝色渐变背景作为开场，建立品牌氛围
- **白色浮动元素**：导航、产品岛悬浮于背景之上
- **Editorial区块**：每个区块有明确主题，标题+描述+展示

### 7.2 信息层级

- **Hero区**：eyebrow → 大标题 → 副标题 → 产品展示
- **Editorial区**：分割线 → 文案 → 视觉卡片
- **数据展示**：大数字（42px）+ 小描述（14px）

### 7.3 品牌元素

- **品牌色**：#2458FF（蓝色）为主品牌色
- **Logo**：32px × 20px 蓝色方块，圆角4px
- **渐变氛围**：蓝色渐变贯穿Hero和重点卡片

## 10. 组件状态总览

### 10.1 交互状态

| 状态 | 视觉表现 |
|------|----------|
| 默认 | 按基础规范显示 |
| hover | 背景加深或文字颜色变化 |
| 激活/选中 | 品牌蓝背景，白色文字 |
| 禁用 | 灰色文字，不可点击 |

### 10.2 数据状态

| 状态 | 视觉表现 |
|------|----------|
| 空状态 | 引导文案 + 操作按钮 |
| 加载中 | 脉冲动画或骨架屏 |
| 成功 | 绿色标识 PASS |
| 失败 | 红/橙色标识，显示原因 |
