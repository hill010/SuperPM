# MVP Scope

## 目标

V1 MVP 的目标是验证「输入剧本 → AI 分镜提示词 → AI 审核 → 人工确认 → 导出」这条核心创作链路是否成立。

MVP 采用本地桌面端、自带 API、本地数据存储，不引入账号、支付、云同步和平台算力。

## MVP 原则

- 先做完整主链路，再做周边增长功能。
- 先做文本生成和审核，再做图片生成。
- 先做本地项目和导出，再做云端资产。
- 先支持「分镜脚本」项目类型，再扩展音乐概念短片、衍生品设计和角色设定。

## V1 必做范围

### 1. 本地桌面应用

| 功能 | 验收标准 |
|---|---|
| 应用启动 | 用户可打开桌面应用并进入首页 |
| 本地项目目录 | 用户可创建并保存项目到本地 |
| 最近项目 | 首页可展示最近创建或打开的项目 |

### 2. 新建项目

| 功能 | 验收标准 |
|---|---|
| 项目名称 | 必填，未填写时不可创建 |
| 项目类型 | V1 默认分镜脚本 |
| 视觉配置 | 支持视觉风格、目标媒介、画幅比例 |
| 默认集数 | 创建项目后自动生成 EP01 |

### 3. 工作台

| 功能 | 验收标准 |
|---|---|
| 三栏布局 | 左栏导航、中栏操作、右栏预览/审核 |
| 集数管理 | 支持添加、切换、删除集数 |
| 阶段步骤条 | 显示四个阶段和当前状态 |
| 阶段锁定 | 前一阶段未确认时，后一阶段不可操作 |

### 4. 四阶段文本工作流

| 阶段 | 输入 | 输出 | 验收标准 |
|---|---|---|---|
| 节拍拆解 | 剧本文本 | 节拍表 | 可生成、预览、审核、确认 |
| 九宫格分镜 | 节拍表 | 9 个分镜提示词 | 可生成、预览、审核、确认 |
| 四宫格序列 | 九宫格提示词 | 36 个序列提示词 | 可生成、预览、审核、确认 |
| 动态提示词 | 全部分镜产物 | Motion Prompt 列表 | 可生成、预览、审核、确认 |

### 5. AI 导演审核

| 功能 | 验收标准 |
|---|---|
| 自动审核 | 每次生成后自动进入审核 |
| PASS/FAIL | 审核结果明确显示状态 |
| 问题列表 | FAIL 时显示问题位置、违反规则和修改建议 |
| 人工确认 | PASS 后用户点击确认才解锁下一阶段 |
| 自动修正上限 | FAIL 后最多自动修正 3 次 |
| 手动处理 | 3 次 FAIL 后允许用户手动编辑并继续审核 |

### 6. 修改意见

| 功能 | 验收标准 |
|---|---|
| 修改输入 | 用户可输入针对当前阶段的修改意见 |
| 重新生成 | AI 根据意见重新生成当前阶段产物 |
| 下游影响提示 | 当前阶段已有下游产物时提示影响范围 |
| 状态标记 | 选择连锁处理时，下游阶段标记为需重新生成 |

### 7. API 配置

| 功能 | 验收标准 |
|---|---|
| 文本 AI 配置 | 可选择服务商并保存 API Key |
| 图像 AI 配置 | 可选择服务商并保存 API Key |
| API 验证 | 可验证 Key 是否可用 |
| 本地加密 | API Key 不明文保存 |
| 未配置提示 | 点击生成时如缺少 Key，提示前往设置 |

### 8. 导出

| 格式 | 验收标准 |
|---|---|
| Markdown | 可导出当前阶段、当前集、全项目 |
| JSON | 可导出结构化数据 |
| 纯提示词文本 | 可按行导出提示词 |

## V1 可选范围

| 功能 | 说明 |
|---|---|
| 单格生图 | 文本工作流稳定后加入 |
| 批量生图 | 单格生图稳定后加入 |
| 模板基础版 | 允许保存和复用视觉配置 |
| 资产库基础版 | 只做保存、查看、下载图片资产 |

## V1 不做范围

| 功能 | 延后原因 | 目标版本 |
|---|---|---|
| 登录/注册 | 需要账号服务和用户系统 | V2 |
| 平台算力 | 需要计费、支付、风控和服务端代理 | V2 |
| 充值页面 | 依赖平台算力 | V2 |
| 云端资产同步 | 需要云存储和账号体系 | V2 |
| 多语言 | 不影响核心链路验证 | V2 |
| PDF 精排导出 | 实现成本高于 Markdown/JSON | V2 |
| 图片 ZIP 导出 | 依赖稳定生图和资产管理 | V2 |
| 团队协作 | 需要权限和同步机制 | V3 |

## V2 方向

- 账号体系：登录、注册、用户设置、语言偏好。
- 平台算力：余额、充值、消耗记录、平台 AI 服务。
- 云端资产：跨设备同步、云端备份、素材复用。
- 项目类型扩展：音乐概念短片、衍生品设计、角色设定。
- 高级导出：PDF 分镜脚本、图片包、视频片段包。

## MVP 验收路径

完整验收必须覆盖以下路径：

```text
启动应用
→ 新建分镜脚本项目
→ 配置 API Key
→ 输入剧本
→ 完成节拍拆解
→ 通过 AI 审核并人工确认
→ 完成九宫格分镜
→ 完成四宫格序列
→ 完成动态提示词
→ 输入修改意见并重新生成
→ 导出 Markdown 和 JSON
→ 关闭应用后重新打开项目，数据完整保留
```
