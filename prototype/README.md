# AI SplitMaster - 交互原型

一个用于验证产品交互流程的 HTML 原型，可直接在浏览器中打开体验。

## 使用方法

1. 用浏览器打开 `index.html` 文件
2. 点击各按钮体验页面跳转

## 包含的页面和交互

### 1. 欢迎页 (Welcome)
- 设置入口（右上角齿轮图标）
- 新建项目按钮
- 最近项目列表（点击可进入工作区）

### 2. 设置页 (Settings)
- 返回按钮
- 文本 AI 服务配置
- 图像 AI 服务配置
- API Key 验证状态

### 3. 工作台 (Workspace)
交互定位：**对话式创作编排**。用户不需要手动管理复杂流程，而是通过自然语言确认方向、提出修改；系统把创意拆成可审核的任务节点，并在画布上沉淀脚本、角色、场景、分镜和音效等产物。

**整体布局：**
- 最左侧轻量导航，可返回首页或新建项目
- 左侧是「创作指挥台」：项目标题、当前 Agent、群聊消息、任务卡和输入框
- 右侧是「节点画布」：点阵背景、创作阶段步骤条、产物节点和画布工具
- 画布左上角只保留创作阶段步骤条，方便用户定位当前阶段

**多 Agent 创作流程：**
- 阶段1「编剧」：生成剧本大纲与节拍拆解
- 阶段2「角色设计师」：生成角色卡片、主视图、三视图和描述
- 阶段3「场景设计师」：生成场景设定、主图描述和多视图四宫格
- 阶段4「分镜师」：确认分镜方案，生成宫格图分镜与 Motion Prompt
- 阶段5「剪辑师（音效）」：生成分镜视频、剪辑节奏、背景音乐、环境音和关键音效方案

**完整成片链路：**
- 基础设置：确认片长、画幅、对白语言和情绪基调
- 剧本生成：形成故事大纲、场景节拍和对白基础
- 角色设计：生成角色卡、主图、三视图和角色资产
- 场景设计：生成关键场景、氛围设定和多视图参考
- 分镜方案：确认分镜策略、视频模型和生成方式
- 宫格分镜：生成可审核的静帧镜头顺序
- 分镜视频：把确认后的宫格分镜生成逐镜头视频片段
- 音效设计：匹配剪辑节奏、配乐、环境音和关键音效
- 最终成片：在画布中展示成品视频预览，并支持导出视频与交付包

**自由画布工作流：**
- 画布助手：理解创作意图、参考素材和输出目标
- 提示词助手：拆解主体、风格、镜头、参数和负面词
- 图像生成器：生成单图、变体和局部重绘结果
- 短视频生成器：生成 3-8 秒视频素材，并沉淀到画布素材区

**角色设计工作流：**
- 角色设定师：整理角色身份、性格、关系和世界观定位
- 造型设计师：生成发型、服装、配色、体态和风格方案
- 角色图生成器：生成主图、三视图、表情和动作参考
- 资产管理员：保存角色资产，供后续故事项目复用

**画布交互：**
- 点击阶段进度可切换当前 Agent
- 点击「确认」会自动进入下一个 Agent 阶段
- 点击「要求修改」会聚焦输入框，继续对当前 Agent 反馈
- 角色、场景、分镜文本支持双击局部编辑
- 分镜宫格支持拖拽排序，并自动更新镜头编号
- 画布节点用于承载阶段产物，后续可扩展为自由拖拽、连线和缩放

### 4. 弹窗
- 新建项目弹窗
- 添加集数弹窗
- 修改意见弹窗
- 导出选项弹窗

## 交互流程验证

### 完整流程
1. 欢迎页点击「新建项目」
2. 输入项目名称 → 进入工作区
3. 在节拍拆解阶段输入剧本 → 点击「开始拆解」
4. 等待 AI 生成 → 查看右侧预览结果
5. 点击「确认通过」→ 自动进入九宫格阶段
6. 重复类似流程完成四宫格和动态提示词
7. 任意阶段可点击「导出」选择格式

### 其他操作
- 点击集数列表切换不同集
- 点击「添加集数」创建新集
- 点击左侧步骤条或底部切换器直接跳转阶段
- 点击「要求修改」提交修改意见后重新生成
- 点击右上角设置图标进入设置页

## 技术说明

- 单文件 HTML，无需构建
- 纯 CSS 实现深色主题
- 原生 JavaScript 实现页面切换和交互
- 模拟 AI 生成过程（2.5秒）
