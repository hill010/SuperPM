# Storyboard Web Visual Reference Notes

## 参考来源

目录：

`I:\12_Vibe Coding\SuperPM\01.AI_SplitMater\前端参考图`

参考图整体来自同一套视觉语言：大面积白/浅灰编辑区、深蓝影像背景、白色浮岛卡片、大圆角媒体容器、超大字号标题、克制的小字号说明和蓝色品牌按钮。

## 可迁移到 Storyboard Web 的设计特征

| 参考特征 | Storyboard Web 转译 |
| --- | --- |
| 白色顶部品牌区 | 登录页和项目页保留白色品牌顶栏，强化 SaaS 信任感 |
| 深蓝/黑蓝大背景 | 登录页、生成预览、素材缩略图和状态画面使用深蓝影像感 |
| 白色浮岛卡片 | 项目卡片、计费弹窗、导出弹窗、权限错误页采用白色浮岛 |
| 大圆角媒体容器 | 首帧/尾帧预览、素材卡、任务状态面板使用 24-44px 大圆角 |
| 超大字号标题 | 仅用于登录/品牌入口，不用于工作台内部，避免影响效率 |
| 灰度分层文字 | 主信息黑/白高对比，辅助说明用灰色，保持参考图的编辑感 |
| 蓝色行动按钮 | 主操作使用 `#2458FF`，与 SplitMaster 视觉系统一致 |

## 不能直接照搬的部分

- 参考图偏品牌/营销展示，Storyboard Web 第一版是工具型 SaaS，工作台内部不能使用大 hero 或低密度排版。
- 参考图里大量装饰性媒体块，Storyboard Web 只能在首帧/尾帧、素材库、登录视觉信号中使用，不能压过镜头编辑信息。
- 参考图的超大标题适合品牌页，不适合镜头列表、任务队列和计费页面。
