# 变更记录

## [v1.0] - 2026-05-02

### 新增
- 新增 Storyboard Web 第一版产品规格，定位为面向外部创作者的 AI 分镜创作 SaaS。
- 新增第一版范围：登录、多用户、个人工作区、云存储、权限、计费、文本拆镜、分镜编辑、首尾帧图生成和导出。
- 新增第二版范围：视频生成、TTS、最终合成、剪辑导出和团队协作。
- 新增计费策略：免费试用 + 订阅 + 积分包。
- 新增权限策略：第一版个人账户权限，数据库预留团队/工作区结构。
- 新增技术方向：Next.js 前端、ASP.NET Core Web API 后端、PostgreSQL、对象存储、任务队列和实时状态推送。
